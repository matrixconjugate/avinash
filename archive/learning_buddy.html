Learning Buddy Project

This project uses Google Apps Script to create a personal learning assistant that helps with enquiry-based learning and spaced repetition. It takes a .txt file and your reading pace as input, makes API calls to the Gemini 1.5 Flash API to generate questions based on the text, compiles the questions and answers in a Google Sheet, and sends you recent and old questions via email for effective learning.

Key Features

  • Accepts .txt files and reading pace as input.
  • Generates questions using Gemini 1.5 Flash API based on the text content.
  • Compiles questions and answers in a Google Sheet.
  • Sends recent and old questions via email for spaced repetition and enquiry-based learning.
